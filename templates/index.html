Welcome to the Qualia Annotation Tool!

The purpose of this tool is to judge whether a given relation between two FrameNet lexical units automatically extracted from an external source is feasible from the human point of view.
The annotation tool provides the following functionality:
  • Annotate a random example
  • View annotations made so far
  • View all available examples
In addition, clicking on a relation ID in table view will bring to the annotation screen for that specific relation.
NB: Currently, < Head, Relation, Tail > should be interpreted the following way: Head's `relation type` is Tail. For example, Head = banquet, Relation = hypernym, Tail = dinner means that banquet's hypernym is dinner.
